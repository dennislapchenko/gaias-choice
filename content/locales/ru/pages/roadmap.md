Мы строим Gaia's Choice в открытую — со всеми незавершёнными частями на виду, а
не только с готовыми. Это честное состояние проекта и то, куда он движется.
Мышление, стоящее за каждой фазой, живёт в нашем бесплатном курсе для
основателей — [«Честный сайт с нуля»](/guides), начиная с
[главы 1 — Плейбука запуска](/guides/kickstart-playbook).

Два правила чтения: **месяцы считаются от первого настоящего обзора**, а не от
сегодняшнего дня, — и фаза открывается только тогда, когда строка «готово,
когда» предыдущей стала правдой. Одна фаза за раз.

## Фаза 0 — Фундамент *(сейчас)*

**Цель: ничего на сайте не фальшиво.**

- [x] Каркас сайта: контент-ориентированный SPA, контейнеризирован, разворачиваемый
- [x] Написан курс для основателей (плейбук, обзоры, трафик, деньги, запуск)
- [x] Страницы раскрытия и конфиденциальности
- [x] Правдивая страница «О нас» — только настоящие факты (история с кемпером, как выяснилось, всё это время была нашей)
- [ ] Пересказать «О нас» своими словами — факты правдивы, но голос пока не наш
- [ ] Один раз и рано решить вопрос публичной идентичности: имена? лица?
      сколько рассказывать про мечту об усадьбе?
- [ ] Реальный контактный email и аккаунты в соцсетях в `site.yaml` — или убрать
      заглушки, пока их нет
- [ ] Купить первые 5–10 продуктов; завести файл заметок о тестировании каждого
- [ ] Домен куплен и привязан

**Готово, когда:** незнакомец может прочитать любую страницу, и его не введут
в заблуждение ни единым словом.

## Фаза 1 — Реальный контент, тихий запуск *(первые ~2 месяца)*

**Цель: 10 настоящих обзоров, ноль заглушек — и тихий запуск.**

- [ ] Опубликован первый настоящий обзор (свой продукт, свои фотографии, честные недостатки)
- [ ] Недельный ритм публикаций выдержан 8 недель подряд
- [ ] Все обзоры-заглушки и ИИ-изображения удалены или заменены
- [ ] 10 обзоров + 2–3 гайда из прожитого дорожного опыта
- [ ] Дописаны оба курса [Пути](/guides) (гомеопатия и травник, главы 4–11,
      на обоих языках) — настоящий контент, который не обзоры

**Ветка разработки (делает контент находимым):**

- [ ] `<title>` и meta-описание для каждой страницы
- [ ] Пререндеринг каждого маршрута в статический HTML на этапе сборки *(ключевая
      задача — открывает SEO, превью Open Graph, Pinterest Rich Pins)*
- [ ] `sitemap.xml`, генерируемый из `content/` на этапе сборки
- [ ] Search Console подтверждён, карта сайта отправлена
- [ ] Аналитика, дружелюбная к приватности — без cookie и баннера согласия;
      сначала обновить [/privacy](/privacy) — она ровно это и обещает

**Готово, когда:** [чек-лист запуска](/guides/launch-checklist) зелёный и тест
тихого запуска пройден.

## Фаза 2 — Трафик *(месяцы 3–6)*

**Цель: сайт читают люди, которых мы не знаем.**

- [ ] Бизнес-аккаунт Pinterest; шаблоны пинов; каждый материал запинен
- [ ] 25+ опубликованных материалов, всё ещё узко, всё ещё еженедельно
- [ ] Один старый материал обновляется каждый месяц — свежесть на существующих
      URL часто обгоняет новый URL
- [ ] Сбор email работает, пусть даже голая форма подписки (решение для
      разработки: бесплатный хостинг-тариф против самостоятельного Listmonk)
- [ ] Первый бесплатный печатный материал как магнит для подписки (чек-лист —
      спутник курса напрашивается первым)
- [ ] Ежемесячные полчаса с метриками: запросы Search Console → контент-план
      следующего месяца; выяснить, какой язык на самом деле приводит читателей,
      и опереться на него

**Готово, когда:** 1000 органических визитов в месяц.

## Фаза 3 — Первый доход *(месяцы 6–12)*

**Цель: сайт честно оплачивает собственный хостинг.**

- [ ] Подать заявки в партнёрские программы — сначала бренды, которые мы уже
      обозрели, затем Amazon Associates *(только когда трафик способен дать 3
      продажи за 180 дней — см. [главу 4 — честную математику](/guides/how-this-site-will-make-money))*
- [ ] Партнёрские ссылки добавлены в существующие обзоры, с раскрытием на каждой странице
- [ ] Опубликованы 2–3 обзора более дорогих товаров (именно они несут математику выручки)
- [ ] Ветка разработки: сервис редиректа `/go/<slug>` со счётчиком кликов, где
      правила программ это допускают — одно место для управления всеми партнёрскими URL

**Готово, когда:** первая комиссия (вставить её в рамку — заслуженно), затем
стабильные $100 в месяц.

## Фаза 4 — Накопление эффекта *(год 1+)*

**Цель: каждый актив кормит другой.** Открывается только когда строка Фазы 3
стала правдой.

- [ ] Платные наборы печатных материалов (чек-листы, списки вещей, спутники курсов)
- [ ] Рассылка как самостоятельный продукт (ежемесячная, по-настоящему полезная)
- [ ] Прямые партнёрства с брендами (раскрытые, без согласования текста — никогда)
- [ ] Пусть альманах ведёт сезонные материалы — настоящей эфемериды нет ни у
      одного конкурента в этой нише
- [ ] Решение по медийной рекламе *(вероятно, нет — плохо вяжется с «покупай меньше»)*
- [ ] Отправить курс для основателей на покой — глава за главой, каждая
      пересказана из прожитого опыта; оригиналы в архив, в уголок «строим в
      открытую»

**Готово, когда:** никогда — эта фаза и есть рабочее состояние, к которому мы
идём.
